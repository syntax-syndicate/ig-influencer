To create a **gray mask over the face** for inpainting with a face reference in ComfyUI's Qwen-Image-Edit workflow (compatible with 2025-2026 updates like Qwen Edit 2511/2509), use the built-in **Mask Editor** on a Load Image node for manual masking, then connect it to the Qwen inpaint nodes.[1][2][3]

### Step-by-Step Workflow Setup
1. **Load the Qwen-Image-Edit Workflow**:
   - Download a recent Qwen-Image-Edit workflow (e.g., from ComfyUI examples or tutorials for 2509/2511 models).[2][6]
   - Drag it into ComfyUI. Ensure models like Qwen-Image-Edit-20B (or GGUF variants) are installed via ComfyUI Manager.[2][5]

2. **Prepare Input Images**:
   - Add a **Load Image** node for your base image (the one to edit).
   - Add another **Load Image** node for your face reference image (used as inpaint guidance via prompt or reference embedding).[1][7]

3. **Create the Gray Face Mask (Manual Method for Precision)**:
   - Right-click the **Load Image** node for the base image.
   - Select **Open in Mask Editor** from the context menu.[1][2][3][4][5]
   - In the editor:
     - Use the brush tool to paint a **gray mask** precisely over the face area (white/gray = masked region for inpainting; black = protected). Cover the full face, including edges, without holes.[1][2][3]
     - Adjust brush size/opacity for feathering if needed (ensures smooth blending).
     - Click **Save** to output the mask.
   - This generates a **grayscale mask** (0=black/protected, 255=white/full inpaint) directly usable by Qwen.[2][5]

4. **Connect for Face Inpainting with Reference**:
   - Wire the **MASK** output from Load Image to the Qwen-Image-Edit node's **mask** input (often via an "Inpaint Crop" or "Embed Crop" node).[1][3][4]
   - Set the **prompt** like: "replace face with reference face, high detail, realistic" (reference the face image via IPAdapter or direct conditioning if workflow supports).[1][7]
   - Optional: Enable **auto-grow mask** (increase "grow value" slider) to expand slightly for better coverage, or use GroundingDINO for initial face detection before manual edit.[1][4]
   - Connect face reference image to a conditioning node (e.g., for face swap enhancement).[1][7]
   - Queue prompt. Qwen will inpaint only the masked face area using the reference.[1][2]

### Tips for Best Results (2025-2026 Workflows)
- **Single vs. Group Faces**: Manual mask excels for group photos (select one face).[1][2]
- **Gray Mask Nuances**: Mid-gray values (e.g., 128) enable partial inpainting for subtle blending; test with denoise strength 0.6-0.8.[3]
- **Alternatives**: Use prompt-based masking (e.g., "face" in GroundingDINO node) then refine manually.[4]
- **Debug Common Issues**:
  | Issue | Fix |
  |-------|-----|
  | Mask too small | Increase grow value or re-brush in editor.[1] |
  | Poor face match | Add face embedding (e.g., via Reactor or IPAdapter nodes).[1][7] |
  | Artifacts | Feather mask edges; use Lama for cleanup.[7] |
