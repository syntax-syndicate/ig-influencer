### Trigger Token Placement and Best Practices
- **Placement:** Always **at the start** of every caption to activate the LoRA reliably during inference (e.g., "trigger_word, [details]"). Add it before or after auto-generated captions if using tools like Next Diffusion or ComfyUI.[1][2][5]
- **Choosing trigger:** Unique name or rare word (e.g., character name, "k1r4lun3z"); use consistently across dataset.[2][5]
- **Length:** Concise (under 75 tokens) but descriptive – simple for beginners ("1girl, red hair"), detailed for advanced.[1]
- **Tools for generation:** Pair images with .txt files (same filename, e.g., "1.png" + "1.txt"); use Florence-2/Clip Interrogator in ComfyUI, or AI like JPT for batch captions, then insert trigger.[2][4][5]
- **Avoid:** Overly long/repetitive details, inconsistent terms, or complex prompts – model infers from patterns.[3]
